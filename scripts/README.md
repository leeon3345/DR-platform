# scripts

개발, 배포 보조, 산출물 생성에 사용하는 스크립트 모음입니다.

## 구조

| 파일 | 역할 |
| --- | --- |
| `build_final_report_docx.py` | 최종 보고서 DOCX 생성 |
| `build_final_report_pdf.py` | 최종 보고서 PDF 생성 |
| `install-k8s-main-local-path.sh` | 실습 클러스터 local-path storage 설정 |
| `patch-my-cluster-agent.sh` | 특정 agent 설정 보정 |
| `recover-agent-identities.sh` | agent identity 복구 보조 |
| `zrok-share.sh` | zrok 공유 URL 설정 보조 |

운영 명령이라기보다 실습 환경 준비와 산출물 재생성을 위한 도구입니다.
