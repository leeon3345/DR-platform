# tools

제품 실행 경로에 직접 포함되지 않는 진단, 유지보수, 실험용 유틸을 모아둔 폴더입니다.

## 구조

| 경로 | 역할 |
| --- | --- |
| `diagnostics/` | API endpoint와 대시보드 상태 점검 도구 |
| `maintenance/` | 과거 코드 수정/정리용 스크립트 |
| `mjs/` | Node.js 기반 실험/진단 스크립트 |
| `misc/` | 분류하기 애매한 실험 파일 |

이 폴더의 파일들은 운영 코드가 아니라 개발자가 문제를 재현하거나 상태를 점검할 때 사용하는 보조 도구입니다.
