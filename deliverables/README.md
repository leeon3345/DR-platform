# deliverables

최종 제출 산출물을 보관하는 폴더입니다.

## 구조

| 경로 | 역할 |
| --- | --- |
| `DR-Platform_v1_Final_Report.docx` | 최종 보고서 원본 |
| `DR-Platform_v1_Final_Report.pdf` | 최종 보고서 PDF |
| `DR-Platform_v1_Final_Presentation.pptx` | 발표 자료 |
| `professor-expected-qna.md` | 예상 질의응답 정리 |
| `artifact-build-manifest.json` | 산출물 생성 메타데이터 |
| `pdf_pages/` | PDF 검수용 페이지 이미지 |

이 폴더는 코드 실행 경로가 아니라 제출물과 검수 결과를 추적하기 위한 공간입니다.
